# MarketGen — симулятор рынка

**MarketGenerator** — класс для генерации синтетических финансовых временных рядов на основе стохастических моделей.  

Симулирует поведение рынка для анализа стратегий и моделирования динамики цен.

---

## Параметры конструктора

| Параметр    | Тип     | Описание                                                 |
| ----------- | ------- | -------------------------------------------------------- |
| **s0**      | `float` | Начальная цена актива                                   |
| **n_steps** | `int`   | Количество временных шагов (например, 252 торговых дня) |
| **n_paths** | `int`   | Количество генерируемых траекторий/симуляций.            |


## Модели

| Модель | Функция | Параметры |
|--------|:-----------:|------------|
| Geometric Brownian Motion (GBM) | `gbm` | `mu` - дрейф, `sigma` - волатильность. `seed` - сид для генератора случайных чисел|
| Random Walk (RW)| `random_walk` | `mu` - дрейф, `sigma` - волатильность. `seed` - сид для генератора случайных чисел|
| Процесс Орнштейна–Уленбека (OU) | `ornstein_uhlenbeck` | `mu` - средний, уровень, к которому стремится процесс, `theta` - скорость стремления к среднему, `sigma` - волатильность, `seed` - сид для генератора случайных чисел|
| Модель Мертона / Jump Diffusion (JD) | `jump_diffusion` | `mu` - дрейф базового GBM, `sigma` - волатильность базового GBM, `lamb` - интенсивность скачков, `mu_j` - среднее логарифмическое изменение при скачке, `sigma_j` - стандартное отклонение логарифмического изменения при скачке, `seed` - сид для генератора случайных чисел|
| Модель Хестона (Stochastic Volatility) | `heston` | `mu` — дрейф, `v0` — стартовая волатильность, `kappa` — скорость возврата, `theta` — средняя волатильность, `xi` — vol of vol, `rho` — корреляция, `seed` - сид  для генератора случайных чисел|
