# MarketGen — симулятор рынка

**MarketGen** — класс для генерации синтетических финансовых временных рядов на основе стохастических моделей.  

Симулирует поведение рынка для анализа стратегий и моделирования динамики цен.

---

## Параметры конструктора

| Параметр    | Тип     | Описание                                                 |
| ----------- | ------- | -------------------------------------------------------- |
| **s0**      | `float` | Начальная цена актива                                   |
| **n_steps** | `int`   | Количество временных шагов (например, 252 торговых дня) |
| **n_paths** | `int`   | Количество генерируемых траекторий/симуляций.            |


## Модели

| Модель | Функция | Параметры |
|--------|:-----------:|------------|
| Geometric Brownian Motion (GBM) | `gbm` | `mu` - дрейф, `sigma` - волатильность. `seed` - сид для генератора случайных чисел|
|Random Walk (RW)| `random_walk` | `mu` - дрейф, `sigma` - волатильность. `seed` - сид для генератора случайных чисел|
